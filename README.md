### h3 How to use API?
---------------------

Follow these steps to get this project live

git clone https://github.com/Mikhalych-KRSK/Laravel-Blog-API
cd Laravel-Blog-API
composer install


Configure your .env file

DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_name
DB_USERNAME=db_username
DB_PASSWORD=db_password
